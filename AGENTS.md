# DocBase Marketplace

DocBase CLI skill for AI agents.

## Skills

- `plugins/docbase-cli/skills/docbase-cli/SKILL.md` — Search, read, create, and manage DocBase memos, comments, users, groups, tags, and attachments via CLI.

## Prerequisites

- [DocBase CLI](https://www.npmjs.com/package/@krayinc/docbase-cli) (`npm install -g @krayinc/docbase-cli`)
- Authenticated with DocBase CLI (`docbase auth login`)
