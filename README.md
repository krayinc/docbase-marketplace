# DocBase Marketplace

[DocBase](https://docbase.io)をAIエージェントから活用するためのマーケットプレイス

## インストール

### DocBase CLI スキル

[DocBase CLI](https://www.npmjs.com/package/@krayinc/docbase-cli)でメモ・コメント・ユーザー・グループ等を操作するためのスキル

#### Claude Code

```
/plugin marketplace add krayinc/docbase-marketplace
/plugin install docbase-cli@docbase-marketplace
```
